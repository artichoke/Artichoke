Macro const_assert_eq

Source 
macro_rules! const_assert_eq {
    ($x:expr, $y:expr $(,)?) => { ... };
}
Expand description

Asserts that two expressions are equal to each other (using PartialEq) at compile time.

See also const_assert!.

§Examples

qed::const_assert_eq!("Veni, vidi, vici".len(), 16);

The following fails to compile because the expressions are not equal:

ⓘ
qed::const_assert_eq!("Carpe diem".len(), 100);
